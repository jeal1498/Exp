# Sistema de Gestión de Expedientes Clínicos — Neuropsicología

Este proyecto implementa un sistema digital de expedientes clínicos para una neuropsicóloga, construido con **Next.js (App Router)**, **Supabase** y desplegado en **Vercel**.

El sistema cumple con las normativas mexicanas vigentes:
- **NOM-004-SSA3-2012**: Del Expediente Clínico.
- **NOM-024-SSA3-2012**: Sistemas de información de registro electrónico para la salud.
- Marco regulatorio: **COFEPRIS**.
